【Sublime Text3 & Java 环境搭建】

原创内容，转载请注明出处！

一、打开用户配置文件夹

  • 菜单栏 ——> 首选项 ——> 浏览插件目录
  • 打开目录下的 User 文件夹

二、配置 Java 环境

编写一个 JavaC.sublime-build 文件放入 User 中

三、编译 JavaC

  • 快捷键 Shift + Ctrl + B
  • 在弹窗中选择 JavaC

四、运行 JavaC - Run

  • 快捷键 Shift + Ctrl + B
  • 在弹窗中选择 JavaC - Run
