### 文档阅读说明

为保证文档内容高质量的展示效果，部分 Markdown 文档中经常会包含许多的高分辨率图片及 GIF 动态图片，并且它们的大小有的会超过 5 MB 甚至更高，目前互联网上暂未找到免费的图床平台能完美的满足要求（容量、速度），综合考虑我将每个 Markdown 文档的图片都集中放置在同一级目录下的 `mark-img` 文件夹中，并在 Markdown 文档中以相对路径的方式引入图片。
